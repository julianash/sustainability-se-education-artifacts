# Sustainability Software Engineering Education Artifacts

This repository contains anonymized instructional artifacts for a project-based Software Engineering education experience focused on sustainability-oriented requirements.

The materials are intended to support transparency, review, and possible adaptation by other educators. They include activity descriptions, classroom templates, review instruments, UML modeling instructions, presentation and reflection prompts, and a sustainability-oriented requirements template.

Student-produced artifacts are not included in their original form because they may contain contextual information related to students, stakeholders, projects, or institutional settings. Examples, when provided, are anonymized, generalized, or synthetic.
